张量算法

\n

一. 张量算法的基本性质

\n
\n

性质：给定具有相同形状的任意张量，任何按元素二元运算的结果都将是相同形状的张量

\n

代码：

\n
from turtle import clone\nimport torch\nx = torch.arange(24).reshape(2,3,4)\ny = x.clone()   #通过分配新内存，将x的一个副本分配给y\nprint(x + y) #同形状张量相加\nprint(x * y) #同形状张量相乘\nprint(x + 2) #张量加常数\nprint(x * 2) #张量乘常数\n
\n

输出：

\n
tensor([[[ 0,  2,  4,  6],\n   [16, 18, 20, 22]],\n\n  [[24, 26, 28, 30],\n   [32, 34, 36, 38],\n   [40, 42, 44, 46]]])\ntensor([[[  0,   1,   4,   9],\n   [ 16,  25,  36,  49],\n   [ 64,  81, 100, 121]],\n\n  [[144, 169, 196, 225],\n   [256, 289, 324, 361],\n   [400, 441, 484, 529]]])\ntensor([[[ 2,  3,  4,  5],\n   [ 6,  7,  8,  9],\n   [10, 11, 12, 13]],\n\n  [[14, 15, 16, 17],\n   [18, 19, 20, 21],\n   [22, 23, 24, 25]]])\ntensor([[[ 0,  2,  4,  6],\n   [ 8, 10, 12, 14],\n   [16, 18, 20, 22]],\n\n  [[24, 26, 28, 30],\n   [32, 34, 36, 38],\n   [40, 42, 44, 46]]])\n
\n

ps：全都是对应元素的运算

\n
\n

二. 降维

\n
    \n
  • 元素/分量 求和
    • \n
\n
\n

方法：调用sum()函数

\n

代码：

\n
from turtle import clone\nimport torch\nx = torch.arange(24, dtype = torch.float32).reshape(2,3,4)\nprint(x.sum())\n
\n

输出：

\n
tensor(276.)\n
\n

ps: 可以给任意形状的张量求和，默认情况下，调⽤求和函数会沿所有的轴降低张量的维度，使它变为⼀个标量。我们还可以指定张量沿哪⼀个轴来通过求和降低维度

\n
\n

轴0：所有的行元素

\n

轴1：所有的列元素

\n

ps: 张量中轴0为竖着的，轴1为横着的

\n
\n

代码：

\n
from turtle import clone\nimport torch\nx = torch.arange(6, dtype = torch.float32).reshape(3,2)\nprint(x)\nprint(x.sum(axis = 0))\nprint(x.sum(axis = 1))\nprint(x.sum(axis = [0,1])) #沿着行和列进行求和等同于整体求和\n
\n

输出：

\n
tensor([[0., 1.],\n  [2., 3.],\n  [4., 5.]])\ntensor([6., 9.])\ntensor([1., 5., 9.])\ntensor(15.)\n
\n
\n
    \n
  • 平均值(mean或average)
    • \n
\n
\n

在代码中，可以调用函数来计算任意形状张量的平均值

\n

代码：

\n
import torch\nx = torch.arange(6, dtype = float32).reshape(2,3)\nprint(s.mean())\nprint(s.sum() / s.numel())\n
\n

输出：

\n
tensor([[0., 1.],\n  [2., 3.],\n  [4., 5.]])\ntensor(2.5000)\ntensor(2.5000)\n
\n

ps：我们可以指定轴进行降维

\n

代码：

\n
import torch\nx = torch.arange(6, dtype = float32).reshape(2,3)\nprint(s.mean(axis = 0))\nprint(s.sum(axis = 0) / s.shape[0]) #注意这里的[]\n
\n

输出：

\n
tensor([[0., 1.],\n  [2., 3.],\n  [4., 5.]])\ntensor([2., 3.])\ntensor([2., 3.])\n

三. 非降维求和

\n

有时候在调用函数来计算总和或者平均值时轴数保持不变会很有用

\n
\n

方法：通过传入参数使轴数不变

\n

代码：

\n
import torch\nx = torch.arange(6, dtype = torch.float32).reshape(3,2)\nprint(x)\nprint(x.mean(axis = 1, keepdim = True))\n
\n

输出：

\n
#True\ntensor([[0., 1.],\n     [2., 3.],\n     [4., 5.]])\ntensor([[0.5000],\n     [2.5000],\n     [4.5000]])\n#False\ntensor([0.5000, 2.5000, 4.5000])\n
\n
\n

例如，由于sum_A在对每⾏进⾏求和后仍保持两个轴，我们可以通过⼴播将A除以sum_A

\n
\n

代码：

\n
import torch\nx = torch.arange(6, dtype = torch.float32).reshape(2,3)\nsum_x = x.sum(axis = 1, keepdim = True)\nprint(x / sum_x)\nprint(sum_x.size())\nprint(sum_x.shape)\nprint(sum_x.numel())\n
\n

输出：

\n
tensor([[0.0000, 0.3333, 0.6667],\n  [0.2500, 0.3333, 0.4167]])\ntorch.Size([2, 1])\n2\n
\n
\n

如果我们想沿某个轴计算A元素的累积总和，比如axis=0（按行计算），我们可以调用cumsum函数。此函数不会沿任何轴降低输⼊张量的维度。

\n
\n

代码：

\n
import torch\nx = torch.arange(6, dtype = torch.float32).reshape(2,3)\nprint(x.cumsum(axis = 0))\n
\n

输出：

\n
tensor([[0., 1.],\n  [2., 3.],\n  [4., 5.]])\ntensor([[0., 1.],\n  [2., 4.],\n  [6., 9.]])\n#层层叠加：0 = 0， 0 + 2 = 2， 0 + 2 + 4 = 6\n
\n

四. 点积

\n

给定两个向量x, y ∈ Rd，它们的点积（dot product）x⊤y （或⟨x, y⟩）是相同位置的按元素乘积的和：x⊤y = ∑id=1 xiyi。

\n
\n

代码：

\n
import torch\nx = torch.arange(6, dtype = torch.float32)\ny = torch.ones(6, dtype = torch.float32)\nprint(torch.dot(x, y))\n
\n

输出：

\n
tensor(15.)\n
\n

ps:

\n
\n

点积在很多场合都很有⽤。例如，给定⼀组由向量x ∈ Rd表⽰的值，和⼀组由w ∈ Rd表⽰的权重。x中的值根据权重w的加权和，可以表⽰为点积x⊤w。当权重为⾮负数且和为1（即(∑id=1 wi = 1)）时，点积表⽰加权平均（weighted average）。将两个向量规范化得到单位⻓度后，点积表⽰它们夹⻆的余弦

\n
\n
\n

五. 矩阵-向量积

\n
\n

ps:注意矩阵A和X的维数(长度)必须相同

\n
\n

代码：

\n
import torch\n\na = torch.arange(20, dtype = torch.float32).reshape(5,4)\nb = torch.ones(4, dtype = torch.float32)\nprint(a)\nprint(a.shape)\nprint(b.shape)\nprint(torch.mv(a,b))\n
\n

输出：

\n
tensor([[ 0.,  1.,  2.,  3.],\n  [ 4.,  5.,  6.,  7.],\n  [ 8.,  9., 10., 11.],\n  [12., 13., 14., 15.],\n  [16., 17., 18., 19.]])\ntorch.Size([5, 4])\ntorch.Size([4])\ntensor([ 6., 22., 38., 54., 70.])\n

六. 矩阵-矩阵乘法

\n
\n

我们可以将矩阵-矩阵乘法AB看作是简单地执⾏m次矩阵-向量积，并将结果拼接在⼀起，形成⼀个n × m矩阵。在下⾯的代码中，我们在A和B上执⾏矩阵乘法。这⾥的A是⼀个5⾏4列的矩阵，B是⼀个4⾏3列的矩阵。两者相乘后，我们得到了⼀个5⾏3列的矩阵。

\n
\n
\n

代码：

\n
import torch\nA = torch.arange(20, dtype = torch.float32).reshape(5,4)\nB = torch.ones(4,3)\nprint(torch.mm(A,B))\n
\n

输出：

\n
tensor([[ 6.,  6.,  6.],\n  [22., 22., 22.],\n  [38., 38., 38.],\n  [54., 54., 54.],\n  [70., 70., 70.]])\n
\n

ps：矩阵-矩阵乘法可以简单地称为矩阵乘法，不应与Hadnmard积混淆

七. 范数

\n
\n

线性代数中最有用的⼀些运算符是范数（norm）。非正式地说，一个向量的范数告诉我们⼀个向量有多大。这里考虑的大小（size）概念不涉及维度，而是分量的大小

\n

在线性代数中，向量范数是将向量映射到标量的函数f。给定任意向量x，向量范数要满⾜⼀些属性。

\n
    \n
  • 范数要满足的一些属性：
    • \n
\n
\n
    \n
  • \n

    第⼀个性质是：如果我们按常数因⼦α缩放向量的所有元素，其范数也会按相同常数因⼦的绝对值缩放：

    \n

    f(αx) = |α|f(x)

    \n
    • \n
  • \n

    第⼆个性质是我们熟悉的三⻆不等式:
    \nf(x + y) ≤ f(x) + f(y).

    \n
    • \n
  • \n

    第三个性质简单地说范数必须是⾮负的:
    \nf(x) ≥ 0[1]

    \n\n
    • \n
\n
\n
\n
\n

范数的计算：

\n
    \n
  • L2范数的计算：
    • \n
\n

在L2范数中常常省略下标2，也就是说||x|| == ||x||2.在代码中，按照如下方式计算向量的L2范数（模长）

\n

代码：

\n
import torch\nu = torch.tensor([3.0,-4.0])\nb = torch.norm(u)\nprint(b)\n
\n

输出：

\n
tensor(5.)\n
\n

ps:在深度学习中我们更常使用L2范数的平方
